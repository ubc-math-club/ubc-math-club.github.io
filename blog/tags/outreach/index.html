Number Theory

August 16, 2021 · 2 min read

Based on some of the problems from Prof. Dragos Ghioca's problem set 1.

Question 1#

Let a,b∈Na, b \in \mathbb{N}a,b∈N. Show that if gcd⁡(a,b)=lcm(a,b)\gcd(a,b) = \text{lcm}(a,b)gcd(a,b)=lcm(a,b), then a=ba = ba=b.

Question Explanation#

a,ba, ba,b here are natural numbers, i.e. the ones that are used to count things: 1,2,…1, 2, \dotso1,2,…

Now, consider the set of all numbers that divide both aaa and bbb. The maximum of this set is defined to be gcd⁡(a,b)\gcd(a,b)gcd(a,b). Formally: gcd⁡(a,b):=max⁡{x:x∣a and x∣b}\gcd(a,b) := \max \{ x : x | a\ \text{and}\ x | b \}gcd(a,b):=max{x:x∣a and x∣b}.

Similarly, consider the set of all numbers that are divisible by both aaa and bbb. The minimum of this set is defined to be lcm(a,b)\text{lcm}(a,b)lcm(a,b). Formally: lcm(a,b):=min⁡{x:a∣x and b∣x}\text{lcm}(a,b) := \min \{ x : a | x\ \text{and}\ b | x \}lcm(a,b):=min{x:a∣x and b∣x}.

Solution 1
  1. a∣lcm(a,b)a | \text{lcm}(a,b)a∣lcm(a,b) and b∣lcm(a,b)b | \text{lcm}(a,b)b∣lcm(a,b), thus a≤lcm(a,b)a \leq \text{lcm}(a,b)a≤lcm(a,b) and b≤lcm(a,b)b \leq \text{lcm}(a,b)b≤lcm(a,b) (by definition).
  2. gcd⁡(a,b)∣a\gcd(a,b) | agcd(a,b)∣a and gcd⁡(a,b)∣b\gcd(a,b) | bgcd(a,b)∣b, thus gcd⁡(a,b)≤a\gcd(a,b) \leq agcd(a,b)≤a and gcd⁡(a,b)≤b\gcd(a,b) \leq bgcd(a,b)≤b (by definition).
  3. Combine both inequalities together:
  • gcd⁡(a,b)≤a≤lcm(a,b)\gcd(a,b) \leq a \leq \text{lcm}(a,b)gcd(a,b)≤a≤lcm(a,b)
  • gcd⁡(a,b)≤b≤lcm(a,b)\gcd(a,b) \leq b \leq \text{lcm}(a,b)gcd(a,b)≤b≤lcm(a,b)
  1. We are given that gcd⁡(a,b)=lcm(a,b)\gcd(a,b) = \text{lcm}(a,b)gcd(a,b)=lcm(a,b), therefore inequalities are actually equalities:
  • gcd⁡(a,b)=a=lcm(a,b)\gcd(a,b) = a = \text{lcm}(a,b)gcd(a,b)=a=lcm(a,b)
  • gcd⁡(a,b)=b=lcm(a,b)\gcd(a,b) = b = \text{lcm}(a,b)gcd(a,b)=b=lcm(a,b)
  1. Hence, a=ba = ba=b.
